------Github ашиглах зааварчилгаа------

---1. github.com веб хуудас руу орж бүртгэл үүсгээд багийн ахлагч руу бүртгэлтэй email, username-г явуулж төслийн ажлын код дээр бүртгүүлнэ

---2. Git-г өөрийн төхөөрөмж дээр татаж авсны дараа CMD дээр доорх тохиргоог хийнэ

git config --global user.name "Your Name"
git config --global user.email "your.email@example.com"

---3. Кодыг шинээр хуулж авахын тулд кодыг байршуулах directory луу очиж, доорх командыг ажиллуулна

cd "your/directory"     -> зааж өгсөн directory руу шилжинэ
cd ..                   -> өмнөх directory руу шилжинэ
git clone https://github.com/nyambayar0118/oop_lab9.git
cd oop_lab9             -> хуулж авсан кодын directory руу шилжинэ

---4. Git ажиллаж байгаа эсэхийг доорх командыг хийж шалгана

git init -b main

---5. Өөрчлөлт оруулсны дараа өөрчлөлтөө хадгалахдаа шинээр файл оруулсан бол доорх командаар нэмнэ

git add --a

---6. Өөрчлөлтийг comment бичиж хадгалахдаа дараах командыг ажиллуулна

git commit -a -m "your message/comment"

---7. Өөрчлөлтийг github дээрх код дээр оруулахдаа дараах командыг ажиллуулна

git push origin main
